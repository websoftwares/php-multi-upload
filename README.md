# php-multi-upload

## System requirements

- PHP 7+
- MySQL 5.6+
- Linux system?

## Installation

1) Execute the database queries from the sql folder.
2) Install dependencies:

```
php composer.phar install
```

4) Start php web server from root folder (not document root)

```
php -S localhost:8080 -t public/
```
